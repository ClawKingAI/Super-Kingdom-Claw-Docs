# Credits and Acknowledgments

Kingdom Claw Forge is an original synthesis that draws inspiration from two source repositories. This file acknowledges those sources while noting that the codebase itself is an original creation.

## Source Repositories

### Agentic Design Patterns
- **Repository**: https://github.com/evoiz/Agentic-Design-Patterns
- **Author**: Antonio Gulli
- **License**: Educational / MIT for code examples
- **What was used**: Concepts and architectural patterns
  - Prompt chaining (sequential decomposition)
  - Routing (dynamic path selection)
  - Parallelization (concurrent execution)
  - Reflection (self-improvement loops)
  - Tool use patterns
  - Planning and task decomposition
  - Multi-agent coordination concepts
  - Memory management patterns
  - MCP (Model Context Protocol) concepts
  - RAG (Retrieval Augmented Generation) patterns
  - Evaluation and monitoring concepts
  - Guardrails and safety patterns

**Note**: No copyrighted book prose was copied. Pattern concepts were extracted and re-implemented with original code and documentation.

### Agency Agents
- **Repository**: https://github.com/ClawKingAI/agency-agents (fork of msitarzewski/agency-agents)
- **License**: MIT
- **What was used**: Inspiration for specialist agent role definitions
  - Agent prompt structure patterns (frontmatter + sections)
  - Role categorization approach
  - Workflow documentation style
  - Success metrics framing

**Note**: Agent definitions in Kingdom Claw Forge are original creations inspired by the structure and approach, not copies.
