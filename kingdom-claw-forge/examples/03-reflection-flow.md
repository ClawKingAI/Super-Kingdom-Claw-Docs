# Example: Reflection and Refinement

## Scenario
User asks: "Write a compelling hero section for the Kingdom Claw landing page. Make it powerful."

## Analysis
- Task complexity: Medium
- Quality requirement: High
- Pattern: Reflection

## Execution

### Step 1: Generate Initial Draft

```
Draft 1:
"Kingdom Claw helps you automate your business with AI agents."
```

### Step 2: Self-Critique

```markdown
## Critique

### What Works
- Clear value proposition
- Simple to understand

### What Could Improve
- Too generic (sounds like every AI tool)
- No differentiation
- Weak hook
- Missing specifics

### Score: 4/10
### Verdict: Rebuild
```

### Step 3: Iterate

```
Draft 2:
"Eight specialized AI agents work 24/7 — finding leads, sending outreach, building websites. Zero AI token costs."
```

### Step 4: Critique Again

```markdown
## Critique

### What Works
- Specific (8 agents, 24/7)
- Concrete benefits (leads, outreach, websites)
- Differentiator (zero AI costs)

### What Could Improve
- Could be more emotionally resonant
- Add typewriter effect for engagement

### Score: 8/10
### Verdict: Ship with enhancement
```

### Step 5: Final Polish

```
Final:
"Your Own AI Agent System. Eight specialized agents working 24/7 — finding leads, sending outreach, building websites. Zero AI token costs."
+ Typewriter animation cycling through key messages.
```

## Pattern Applied

**Reflection** — Generate, critique, improve, ship when quality threshold met.
