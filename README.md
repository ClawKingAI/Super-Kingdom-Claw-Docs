## 🚀 Quick Deploy

```bash
# Clone
git clone https://github.com/ClawKingAI/Super-Kingdom-Claw-Docs.git
cd Super-Kingdom-Claw-Docs

# Install
npm install -g openclaw

# Configure
cp reference/CONFIG-TEMPLATE.yaml ~/.openclaw/config.yaml
# Add your API keys

# Launch
openclaw gateway start
```
