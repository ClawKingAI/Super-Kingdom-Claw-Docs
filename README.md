## 🚀 Quick Start

### Prerequisites
- Node.js 20+
- Python 3.11+
- VPS or cloud server (2GB RAM minimum)

### Deploy Kingdom Claw

```bash
# 1. Clone this repository
git clone https://github.com/ClawKingAI/Super-Kingdom-Claw-Docs.git
cd Super-Kingdom-Claw-Docs

# 2. Install OpenClaw
npm install -g openclaw

# 3. Configure
cp reference/CONFIG-TEMPLATE.yaml ~/.openclaw/config.yaml
# Edit config.yaml with your API keys

# 4. Start
openclaw gateway start
```

[→ Full deployment guide](deployment/VPS-SETUP.md)

## 🔧 Configuration

### Model Providers

Kingdom Claw supports multiple model providers:

```yaml
providers:
  - name: nvidia
    apiKey: YOUR_NVIDIA_API_KEY
    models:
      - nvidia/z-ai/glm5
      - nvidia/moonshotai/kimi-k2.5
  - name: openai
    apiKey: YOUR_OPENAI_API_KEY
    models:
      - openai/gpt-4.1
```

### Skills

Skills are loaded from the repository:

```yaml
skills:
  - ~/Super-Kingdom-Claw-Docs/skills/kingdom-mastery
  - ~/Super-Kingdom-Claw-Docs/repos/antigravity-awesome-skills/skills
```

## 📖 Key Concepts

### How Agents Work

Kingdom Claw dispatches tasks to specialized agents:

```
Task → Orchestrator → Route to Specialist → Execute → Reflect → Learn
```

### How Skills Work

Skills are modular capabilities:

```
SKILL.md defines:
- When to trigger
- What to do
- How to do it
- What to return
```

### How Evolution Works

The system self-improves:

```
Execution → Analyze → Detect Pattern → FIX/DERIVE/CAPTURE → Update
```
