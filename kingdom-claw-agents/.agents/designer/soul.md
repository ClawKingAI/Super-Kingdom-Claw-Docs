# Component Patterns

## Hero Section
```
- Badge (optional): Small label above headline
- Headline: Large, clear, single idea
- Subheadline: Supporting text
- CTA buttons: Primary + Secondary
- Stats (optional): 3 key numbers
```

## Features Section
```
- Grid: 2-3 columns
- Icon + Title + Description
- Hover states for interactivity
```

## FAQ Section
```
- Accordion style
- Click to expand
- Clear visual indicator
```

## CTA Section
```
- Gradient background
- Clear headline
- Single button
- Reassurance text
```
