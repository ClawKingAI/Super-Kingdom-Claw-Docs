# Client Outreach Workflow

## Purpose
Automated and semi-automated outreach for Kingdom Claw client acquisition.

## Templates

### Discord/Community DM (High Response)
```
Hey [Name]! Saw your message in [Community] about [specific problem they mentioned]. I actually built something similar for [similar use case/client] - we reduced [metric] by [X%] in [timeframe]. Would you be open to a quick chat about what you're trying to build? Happy to share what worked (and what didn't).

[Your name]
```

### Cold Email (B2B Standard)
```
Subject: [Company]'s approach to [specific problem]

Hi [First name],

I noticed [Company] is [specific observation from research - funding, product launch, job posting]. We helped [similar company] implement [solution] and they saw [specific result with numbers].

Would a 15-minute call to explore if something similar could work for [Company] be worth your time?

[Your name]
[One-line credibility]

P.S. [Personal observation about their work/content]
```

### LinkedIn Connection + Message (Enterprise)

**Connection Request:**
```
Hi [Name] - I've been following [Company]'s work in [space] and would love to connect. Your recent [specific content they posted] resonated with [why it resonated].
```

**Follow-up Message (after acceptance):**
```
Thanks for connecting, [Name]! I noticed [Company] is [specific situation]. We've helped companies like [reference] solve [problem] through [approach]. Would it make sense to have a brief call? I'd be happy to share what we've learned, whether or not there's a fit.
```

### Follow-up #1 (Day 4-5)
```
Hi [Name],

Just wanted to bump this up in case it got buried. Quick question: Is [problem I mentioned] something [Company] is actively working on, or is it on the back burner for now?

Either way totally fine - just want to be respectful of your inbox.

[Your name]
```

### Follow-up #2 (Day 10-12)
```
Hi [Name],

One last ping on this. I'll assume now isn't the right time and won't follow up further. If things change, here's a link to my calendar: [Calendar link]

Best,
[Your name]
```

### Referral Request (Post-Project)
```
Hi [Client Name],

Really enjoyed working on [project] with you! Do you know anyone else who might benefit from similar work? I'm looking to connect with [specific profile] who need [specific outcome].

A warm intro goes a long way - happy to offer [discount/referral fee] for any successful introductions.

Thanks again for the great project!

[Your name]
```

## Discovery Call Script

**Opening (2 min)**
```
"Thanks for taking the time. I'd love to understand what you're working on and see if there's a fit. I'll ask some questions, and at the end, I'll let you know honestly if I think I can help - and if not, I'll try to point you in the right direction. Sound good?"
```

**Problem Exploration (10 min)**
```
"Tell me about [problem area]. What's happening now?"
→ Listen. Then: "What have you tried? What worked? What didn't?"
→ "What's the impact of this problem? On revenue? On time? On stress?"
```

**Desired Outcome (5 min)**
```
"If we solved this perfectly, what would that look like?"
"How would you know it was successful? What metrics matter?"
```

**Decision & Budget (5 min)**
```
"What's your timeline for tackling this?"
"What kind of budget have you allocated?"
"Who else needs to be involved in this decision?"
```

**Closing (5 min)**
```
"Based on what you've shared, here's what I'm hearing..."
"I think [can/cannot] help. Here's why..."
"Next step would be [proposal/discovery deeper call/intro to someone else]"
```

**Always End With:**
```
"What questions do you have for me?"
```

## Red Flags (Decline or Proceed with Caution)

| Flag | Meaning | Action |
|------|---------|--------|
| "I need this yesterday" | Unrealistic timeline | Require detailed scope before committing |
| "Can you work for equity?" | No budget | Decline unless exceptional opportunity |
| "I'll know it when I see it" | Scope creep guaranteed | Require detailed requirements |
| "My last developer ghosted me" | May be difficult client | Ask for reference from previous devs |
| "It's a simple project" + vague | Never simple | Force detailed specification |
| Won't share budget | Time-waster | Ask directly 2x, then pass |
| Multiple decision makers | Approval nightmare | Get all parties on kickoff call |
| NDA before discussion | Corporate bureaucracy | Standard NDA ok, custom NDA = red flag |
| Asking for free work | Doesn't value expertise | Decline politely |
| Price-focused only | Will nickel-and-dime | Offer fixed price with clear scope |

## Ideal Client Profiles

### 🚀 Startup Founder (Seed-Series A)
- Technical/semi-technical background
- Building MVP or scaling
- Needs speed over perfection
- Budget: $5K-$50K/project, $3K-$10K/mo retainer

### 🏢 Enterprise Manager (Director/VP, 500+ company)
- Non-technical decision maker
- Budget authority, procurement process
- Risk-averse, needs documentation
- Budget: $50K-$500K/project, 3-6 month sales cycle

### 🎨 Agency Owner (Marketing/Design/Dev)
- Business-minded, understands value
- Has clients needing AI solutions
- Values white-label partnerships
- Budget: $10K-$100K/project, retainer relationships

## Outreach Schedule

| Day | Action |
|-----|--------|
| 0 | Initial outreach (DM/email/LinkedIn) |
| 4-5 | Follow-up #1 |
| 10-12 | Follow-up #2 (final) |
| Post-response | Discovery call within 48 hours |
| Post-call | Proposal within 2-5 days |

## Key Metrics to Track

- Sent: Total outreach messages sent
- Opened: Emails opened (if trackable)
- Responded: Replies received
- Discovery calls booked: From positive responses
- Close rate: Signed contracts / discovery calls
- Referral rate: New intros from existing clients
