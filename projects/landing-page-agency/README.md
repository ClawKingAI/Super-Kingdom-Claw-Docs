# LaunchPad - AI-Powered Landing Pages

A premium, production-ready landing page for an AI-powered landing page agency.

## Features

- 🎨 Dark, premium aesthetic (Stripe/Linear/Vercel level polish)
- ✨ Glass morphism elements with gradient accents
- 🎭 Smooth animations powered by Framer Motion
- 📱 Fully responsive design
- 🔍 SEO optimized with meta tags
- ⚡ Static HTML export for fast deployment

## Tech Stack

- React 18 + TypeScript
- TailwindCSS 3
- Framer Motion
- Lucide Icons
- Vite

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The static site will be output to the `dist/` directory.

## Deployment

The `dist/` folder is ready for deployment to here.now or any static hosting service.
