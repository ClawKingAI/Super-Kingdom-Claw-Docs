## Analysis: Kingdom Claw Adaptation Potential

### High-Value Prompts for Our Workflows

#### 1. **Anti-Gravity / Artifact Deck** (Most Aligned)
Why: Matches Kingdom Claw's aesthetic — minimal, calm, confident, precise
Use case: Client proposals, system documentation, agent workflow visualizations
Key principles:
- Pure white background with soft gradient accents
- Calm authority typography (not aggressive)
- High negative space
- "If it feels inevitable, it's right"

#### 2. **Sharp-Edged Minimalism**
Why: Professional, architectural feel for technical documentation
Use case: API docs, agent capability sheets, workflow diagrams
Key principles:
- Grid-based layout
- Top-left navigation
- Dark mode option for emphasis slides

#### 3. **Modern Newspaper**
Why: Editorial authority for thought leadership content
Use case: LinkedIn carousels, Twitter threads converted to slides
Key principles:
- Swiss Style / Bauhaus inspiration
- Asymmetrical placement
- 10:1 headline-to-body ratio
- "1 slide = 1 message"

#### 4. **Tech / Art / Neon**
Why: Avant-garde, intellectual aesthetic for brand differentiation
Use case: Landing pages, product reveals, capability showcases
Key principles:
- Warm gray background (#E0E0D0)
- Neon yellow accents (#DFFF00)
- Monochrome cut-out portraits
- Architectural draft-line feel

#### 5. **Neo-Retro Dev Deck**
Why: Developer-centric, playful but opinionated
Use case: Technical tutorials, agent architecture explanations
Key principles:
- Grid paper background
- Pixel-art icons
- High-contrast color blocks
- "Builders explaining the future, not selling fluff"

### Not Suitable for Kingdom Claw
- Manga Style (too playful)
- Pink Street-style (wrong demographic)
- Deformed Flat Persona (too casual)
