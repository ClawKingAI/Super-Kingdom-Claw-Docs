## Key Learnings

1. **Docker Architecture:** OpenClaw runs inside Docker container on this VPS. Paths inside container (`/data/.openclaw/`) don't exist on host. Dashboard running on host cannot access container files directly.

2. **User Communication:** User has low tolerance for technical jargon. Requires exact step-by-step commands. Prefers I execute when possible, not instruct.

3. **Dashboard Requirements:**
   - Node.js 20.9.0+ required (Next.js 16 requirement)
   - Reads OpenClaw config from `~/.openclaw/openclaw.json`
   - Can run on any port via `PORT=XXXX npm run dev`
