### Agent Memory System Design
- **Goal:** Agents that learn from previous sessions and pass work between each other
- **Discovery:** Can only spawn generic subagents, not specialist types
- **Solution:** Shared memory files + handoff protocol
- **Structure:**
  ```
  /data/.openclaw/workspace/agent-memory/
  ├── sessions/*.json — Agent state per role
  ├── handoffs/pending/ — Work to be picked up
  ├── handoffs/completed/ — Work that was done
  ├── outputs/**/*.json — Completed work artifacts
  └── lessons-learned.json — Accumulated wisdom
  ```
- **Workflow:** Subagent reads session file → does work → writes output → creates handoff if incomplete
- Main agent monitors handoffs/pending/ and routes to appropriate subagent
- Design doc: `/data/.openclaw/workspace/agent-memory/DESIGN.md`
