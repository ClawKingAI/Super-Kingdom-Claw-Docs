## Key Learnings
1. **Facebook API limitations:** Can only manage Pages, not personal profiles
2. **Scam detection:** Most Page messages were phishing scams pretending to be "Meta Support"
3. **Token types:** User token vs Page token distinction matters - need Page token for posting
4. **Curl redirects:** Public Google Sheets need `-L` flag to follow redirects
5. **App passwords:** Gmail app passwords can expire - verify before assuming failure
6. **Inner Warden patterns:** AST-based command analysis, layered defense, fail-open, advisory tracking
7. **Docker limitations:** Container kills background processes - use here.now for static deploys
8. **CORS architecture:** Browser cannot call external APIs directly - need server-side proxy
9. **Static hosting limits:** No compute = no API proxying = broken for AI apps
10. **Serverless solution:** Next.js API routes on Netlify solve CORS, hide keys, add compute
