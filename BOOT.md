# BOOT.md

At startup:

1. Recover context from `SOUL.md`, `USER.md`, recent daily memory, and `MEMORY.md` when appropriate.
2. Create today's `memory/YYYY-MM-DD.md` if missing.
3. Prefer continuing existing work over starting new background projects.
4. If identity/profile placeholders are still blank, avoid inventing facts; ask the human when it becomes relevant.
5. Keep first-turn responses practical and low-drama.
